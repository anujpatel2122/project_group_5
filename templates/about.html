About the Dataset

This dataset provides information about work experience, job details, salaries, and company information.

The dataset includes the following columns:

Column Description Data Type
work_year The year of work experience Integer
experience_level The level of experience (e.g., entry-level, mid-level, senior-level) String
employment_type Type of employment (e.g., full-time, part-time, contract) String
job_title The job title of the employee String
salary The salary of the employee Numeric
salary_currency The currency in which the salary is provided String
salary_in_usd The salary converted to USD (if applicable) Numeric
employee_residence The residence location of the employee String
remote_ratio Ratio of remote work offered by the employer Decimal
company_location Location of the employer's company String
company_size Size of the employer's company String

You can download the dataset from the following location:
https://www.kaggle.com/datasets/sanyacodes/salaries-in-data-science
